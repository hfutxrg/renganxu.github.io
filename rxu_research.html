Research Experiences
  • OpenUH Compiler: An Open Source Implementation of OpenACC Code Link
    1. Designed and developed the runtime library for an open source OpenACC compiler (OpenUH) using CUDA Runtime API and Driver API.
    2. Explored and implemented OpenACC reduction algorithms and multi-GPU extension.
    3. Debugging OpenUH compiler to fix bugs in compiler back-end.
  • OpenACC Application Suite Code Link
    1. Developed OpenACC applications including NAS Parallel Benchmarks and Rodinia benchmarks.
    2. Contributed four OpenACC NAS benchmarks (EP, CG, SP and BT) to SPEC ACCEL V1.0 (see SPEC ACCEL V1.0).
  • OpenACC 1.0 Validation Suite
    1. Designed and implemented a robust and scalable validation infrastructure to evaluate the correctness of OpenACC 1.0 in compilers.
    2. Evaluated four compilers (CAPS, CRAY, PGI and OpenUH) with this suite.
    3. Integrated the validation suite into the harness infrastructure of the TITAN supercomputer at Oak Ridge National Laboratory.
  • Intern Developer at TOTAL E&P Research and Technology USA
    1. Accelerated the Fortran version of a seismic imaging application called Kirchhoff Migration to many-core GPU architecture using OpenACC.
    2. Analyzed the performance bottleneck with profiling tools and then improved the performance significantly by applying different optimizations.
    3. Investigated the cause of numerical inaccuracy with GPU debugging tool and solved that issue.
  • Intern Developer at Repsol E&P USA
    1. Parallelized the Kirchhoff Demigration application with the hybrid model of MPI and POSIX threads.
    2. Designed and developed a framework to achieve dynamic load balancing in MPI applications.
    3. Applied the same framework for parallel I/O which can reach the theoretical peak performance for parallel read, with Panasas as the underlying parallel file system and RAID 1/5 as the data storage technology.
    4. Proposed and implemented a strategy to solve the fault tolerance issue in MPI applications.
    5. Ported the C++ version of Reverse Time Migration (RTM) application with the hybrid model of POSIX threads, CUDA and MPI onto a GPU cluster to maximize the parallelism and performance.
